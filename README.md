# python-cli

Practice coding a CLI with Python. Interacts with the Spotify API.

## Prerequisites

1. Get your Client ID and Client Secret from your Spotify Dashboard. See docuementation: [Request an access token](https://developer.spotify.com/documentation/web-api/tutorials/getting-started#request-an-access-token)
2. Put the values in the `.env` file at the root of this repository.

    ```toml
    SPOTIFY_CLIENT_ID=your_client_id_here
    SPOTIFY_CLIENT_SECRET=your_client_secret_here
    ```
